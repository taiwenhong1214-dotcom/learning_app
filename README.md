# 🇲🇾 LearnBM (MyBahasa)

**LearnBM** (also known as MyBahasa) is a personalized, intelligent mobile application designed to assist non-native speakers in learning Bahasa Malaysia through structured, interactive, and engaging content. 

Built with **Flutter** and powered by **Firebase** & **AI**, LearnBM bridges the gap for beginners (such as international students, expatriates, and language enthusiasts) by offering a modern, beautiful, and personalized language learning experience.

---

## ✨ Key Features

### 🔐 Secure Authentication
- **Firebase Auth Integration**: Seamlessly log in or register via **Email/Password**, **Google Sign-In**, or jump right in as a **Guest**.
- **Cloud Persistence**: Your learning progress, bookmarks, and quiz scores are securely saved to the cloud and synced across devices.

### 📚 Structured Learning Library
- **6 Thematic Categories**: Explore vocabulary across carefully curated topics: *Keluarga (Family), Kenderaan (Vehicles), Nombor (Numbers), Warna (Colours), Buah-buahan (Fruits), and Haiwan (Animals)*.
- **Rich Context**: Every vocabulary card includes the Bahasa Melayu word, English translation, and example sentences in both languages.
- **Text-to-Speech (TTS)**: Tap to hear the accurate, native pronunciation of any word.

### 🤖 AI-Powered Tools
- **AI Conversation Practice**: Chat with an intelligent AI tutor (powered by advanced LLMs) to practice real-life dialogues. Receive instant grammar corrections, hints, and encouragement.
- **AI Quiz Generator**: Generate dynamic, multiple-choice quizzes on the fly. Choose your difficulty (Easy, Medium, Hard) and let the AI test your knowledge!

### 📈 Progress Tracking & Gamification
- **Personalized Tracking**: Bookmark your favourite words or mark them as "Learned".
- **Global Leaderboard**: Compete with other learners globally! Complete quizzes to earn points and climb the ranks on the live Firebase-powered leaderboard.

## 🛠️ Technology Stack

- **Frontend**: [Flutter](https://flutter.dev/) & Dart (Material Design 3)
- **Backend/Database**: [Firebase](https://firebase.google.com/) (Authentication & Cloud Firestore)
- **AI Integration**: OpenRouter / LLM APIs (DeepSeek, Nemotron, etc.)
- **State Management**: Stateful Widgets & Flutter core mechanics.

## 🚀 Getting Started

To run this project locally, follow these steps:

### Prerequisites
- Flutter SDK (`v3.0.0` or higher)
- Android Studio or VS Code
- A connected Android emulator or physical device.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/taiwenhong1214-dotcom/learning_app.git
   ```
2. **Install dependencies**
   ```bash
   cd learning_app
   flutter pub get
   ```
3. **Run the app**
   ```bash
   flutter run
   ```
